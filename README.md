## Frontend
### Install Node
Go to *nextjs* Directory and install dependencies:

    cd/nextjs
    npm install

# Run Project

## Run Backend
Open Virtual Enviroment:

    ve/Scripts/Activate
Go to *backend* directory:

    cd backend
Run Django:

    python manage.py runserver

## Run Frontend
Go to *nextjs* directory:

    cd/nextjs
Run Frontend:

    npm run dev
